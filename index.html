⚙️ ตั้งค่าระบบ

รูปแบบ: ชื่อ=PIN:สิทธิ์ คั่นด้วยจุลภาค — สิทธิ์ owner เห็นทุกเมนู / staff เห็นเฉพาะออเดอร์+ตั้งค่า เช่น yui=080766:owner, pinky=1234:staff
รองรับ Thai QR ร้านค้าจากแอพธนาคาร เช่น Merchant Pro (กรุงเทพ), แม่มณี (SCB), ถุงเงิน ฯลฯ
ระบบจะบีบอัดรูปก่อนอัปโหลดเข้า Google Drive แล้วเก็บเป็น URL ในชีต Settings (ป้องกันปัญหา cell limit 50,000 อักขระ)
ขั้นตอน: สร้างบอทกับ @BotFather ใน Telegram → วาง Token → กด 💾 บันทึก → เปิด Telegram ทักหาบอท 1 ข้อความ → กลับมากด "หา Chat ID" → บันทึกอีกครั้ง
เมื่อลูกค้าอัปสลิป ระบบจะอ่านยอดเงินในรูป (OCR) — ยอดตรง → แชร์สิทธิ์ผู้อ่านให้ Gmail ลูกค้า + ส่งแผนทางอีเมลทันที แจ้งผลใน Telegram 🤖✅ | ยอดไม่ตรง/อ่านไม่ได้ → ไม่ส่ง ค้างไว้ที่หน้าต้องส่ง และแจ้ง Telegram ให้ตรวจเอง
⚙️ ต้องเปิด Advanced Drive Service ใน Apps Script ก่อน (เมนู Services → Drive API → Add)
